模块化定义

  • 一系列相关联的结构组成的整体
  • 带有一定语义，而非表现

怎么做

  1. 为模块分类命名（如m-,md-）
  2. 以一个主选择器开头（模块根节点）
  3. 使用以主选择器开头的后代选择器（模块子节点）
